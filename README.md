Dental Assistant Award FAQ

A professional, interactive FAQ interface designed for dental assistants to understand their workplace rights under the HPSS Award and National Employment Standards (NES).

🚀 Features

Interactive Accordion UI: Smooth, animated transitions using CSS transitions and JavaScript.

Brand-Compliant Design: Utilizes a sophisticated "Dental Professional" color palette:

Deep Purple (#532A8C) for primary headings and icons.

Muted Purple & Lilac for borders and accents.

Success Green (#7ABF49) for active states.

Light Grey (#F2F2F2) for content backgrounds.

Responsive Layout: Mobile-first design using Tailwind CSS that looks great on desktops, tablets, and phones.

Scenario-Based Content: Covers critical topics like overtime refusal, professional boundaries with patients, and handling wage underpayments.

🛠️ Built With

HTML5: Semantic structure.

Tailwind CSS: Modern utility-first styling.

Google Fonts: "Inter" typeface for high readability.

Vanilla JavaScript: Lightweight logic for accordion interactions.

📂 File Structure

faq.html: A single-file solution containing all HTML, CSS, and JS logic.

💻 How to Use

Local View: Open faq.html in any modern web browser.

Integration: The code is self-contained. You can copy the <style>, <body>, and <script> contents into your existing web platform or CMS.

Customization:

Update the brand colors in the <style> block to match your specific practice.

Add new FAQ items by following the faq-item div structure provided in the source code.

⚖️ Disclaimer

This tool provides general information based on the HPSS Award and NES. It does not constitute legal advice. Users should always consult with the Fair Work Ombudsman or a legal professional for specific workplace disputes.
